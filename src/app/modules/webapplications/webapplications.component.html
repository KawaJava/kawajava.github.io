MMOEstateManager

MMOEstateManager to nowoczesna aplikacja internetowa, która oferuje graczom MMO intuicyjny interfejs do przeglądania i zarządzania włościami w grach, w tym stanem złota oraz danymi o szeryfach i wójtach. Dzięki śledzeniu historycznych zmian, użytkownicy mogą monitorować ilości złota i szybko identyfikować anomalie, np. przekroczenie ustalonych limitów.

Kluczowe funkcje

  • Przeglądanie i edytowanie danych o włościach, w tym stanu złota oraz zarządzania szeryfami i wójtami.
  • Automatyczne wysyłanie powiadomień e-mail do graczy, którzy nie zaktualizowali danych o złocie, eliminując ręczne przypomnienia.
  • Panel administracyjny z interaktywnymi wykresami historycznych zmian.
  • Bezpieczna autoryzacja oparta na JWT Token, z wydzielonymi rolami Admin i User.
  • Generowanie raportów oraz zarządzanie uprawnieniami i danymi wójtów.

Backend

Podstawę systemu stanowi:

  • Java 21
  • Spring Boot 3 (Web, Data-JPA, Liquibase, AOP, Mail Sender, Scheduler, Validation, Security – JWT Token)
  • MySQL
  • REST API
  • Testy jednostkowe (JUnit, Mockito, AssertJ)
  • Testy API i manualne (Postman)
  • Wzorce projektowe (adapter, strategia, builder)
  • Lombok
  • Maven
  • Architektura warstwowa
  • Git

Frontend

  • Angular 13.3: struktura aplikacji, komponenty, routing i lazy loading, umożliwiające dynamiczne ładowanie danych w zależności od potrzeb.
  • TypeScript: bezpieczne typowanie danych zapewniające lepszą jakość kodu.
  • RxJS: zarządzanie asynchronicznością i strumieniami danych, wykorzystujące operatory takie jak BehaviorSubject, tap, czy catchError.
  • Angular Material: zaawansowane komponenty UI (MatTable, MatDialog, MatPaginator, MatSnackBar) zapewniające nowoczesny interfejs użytkownika.
  • Angular Flex-Layout: responsywne layouty dostosowujące się do różnych rozmiarów ekranów.
  • SCSS: elastyczne stylowanie umożliwiające pełną kontrolę nad wyglądem aplikacji.
  • Reactive Forms: zaawansowane zarządzanie formularzami z walidacją i dynamicznymi reakcjami.
  • Angular Router: routing aplikacji i zabezpieczanie ścieżek za pomocą AuthGuard.
  • Interceptors: automatyczne dołączanie tokenów JWT do żądań HTTP oraz obsługa błędów związanych z autoryzacją.
  • Chart.js + ng2-charts: tworzenie interaktywnych wykresów (BarChart, PieChart) do wizualizacji danych.

Technologie

  • Frontend: Angular 13.3, TypeScript, RxJS, Angular Material, Angular Flex-Layout, SCSS, Reactive Forms, Angular Router, Interceptors, Chart.js + ng2-charts
  • Backend: Java 21, Spring Boot 3 (Web, Data-JPA, Liquibase, AOP, Mail Sender, Scheduler, Validation, Security - JWT Token), MySQL, REST API, wzorce projektowe (adapter, strategia, builder), Lombok, Maven, architektura warstwowa, Git
  • Testy: JUnit, Mockito, AssertJ, Postman
